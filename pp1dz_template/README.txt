Pri otvaranju projekta u razvojnom okruzenju Eclipse, 
treba odabrati direktorijum 'workspace' za radni prostor.

Direktorijum 'workspace' se nalazi u istom direktorijumu kao ovaj fajl.
Klasa MJTest je test za leksicki analizator koja se moze pokrenuti 
koriscenjem Java Application konfiguracije izvrsavanja.
